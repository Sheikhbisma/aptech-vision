HTML Headings

<h1>Main Heading</h1>
<h2>Sub Heading</h2>

HTML headings range from <h1> to <h6>. <h1> is the largest.

HTML headings <h1> se <h6> tak hotay hain. <h1> sab se bara hota hai.

CSS Selectors

h1 {
  color: red;
}
#myId {
  font-size: 18px;
}
.myClass {
  background-color: blue;
}

CSS selectors are used to select elements and apply styling to them.

CSS selectors ka use HTML elements ko select karne aur un par style apply karne ke liye hota hai.

Bootstrap Grid System

<div class="container">
  <div class="row">
    <div class="col-md-6">Column 1</div>
    <div class="col-md-6">Column 2</div>
  </div>
</div>

Bootstrap uses a 12-column grid system for layout. You can divide columns using classes like col-md-6.

Bootstrap 12-column grid system use karta hai. Aap col-md-6 jaise classes se columns divide kar saktay hain.

JavaScript Alert

<script>
  alert("Hello World");
</script>

The alert() method in JavaScript displays a popup box with a message.

alert() method JavaScript mein ek popup message box show karta hai.

jQuery Click Event

<script>
  $("button").click(function(){
    alert("Button clicked!");
  });
</script>

This jQuery function runs when any button is clicked, showing an alert message.

Yeh jQuery function tab run hota hai jab koi button click kiya jaye, aur ek alert show karta hai.
